Odoo Enterprise Theme

This module will merely adopt Odoo Enterprise theme. This module will be used for that User who use the Odoo Community.

Key Features:

  • Modern Light Design: Professional light color scheme matching Odoo 17, 18, and 19 Enterprise editions with clean white backgrounds, subtle borders, and the iconic purple brand color (#875A7B) for a polished, contemporary look.
  • Signature Purple Branding: Features Odoo's distinctive purple accent (#875A7B) for active tabs, buttons, and brand elements, maintaining visual consistency with Odoo Enterprise identity while using the modern light interface paradigm.
  • Clean Component Styling: Applies refined styling to navbar, forms, lists, tables, tabs, and buttons with white/light grey backgrounds, subtle shadows, proper spacing, and crisp borders for enhanced readability and professional appearance.
  • Optimized Readability: Uses light backgrounds (#ffffff, #f9f9f9) with dark text (#666666, #212529) for maximum contrast, reducing eye strain during extended work sessions and improving text clarity across all screen types.
  • One-Click Installation: Simple module installation with automatic SCSS compilation, works seamlessly with Odoo 16/17/18/19 Community editions, requires no manual configuration, and maintains full compatibility with all standard Odoo features.

Frequently Asked Questions

Is this the same as Odoo 16 Enterprise theme or a different version?

This is a modern LIGHT theme inspired by Odoo 17/18/19 Enterprise, NOT the dark Odoo 16 theme. While Odoo 16 Enterprise had a dark purple background, newer Odoo versions (17/18/19) use clean white backgrounds with purple accents. This theme matches the newer light design while keeping the signature purple brand color (#875A7B) that makes Odoo recognizable.

Why does this theme look different from standard Odoo Community Edition?

Standard Odoo Community Edition has a basic blue/grey interface. This theme replicates the premium Enterprise look with purple branding (#875A7B), refined spacing, cleaner borders, better button styling, and professional color schemes used in Odoo's paid Enterprise versions. It gives Community Edition users the polished Enterprise appearance without the Enterprise license cost.

Does the purple color (#875A7B) apply to all buttons or only specific ones?

The purple accent applies selectively to maintain visual hierarchy:

Follow-up Features:
  • Active tab borders (top border when selected)
  • Primary/brand elements (logo area, key actions)
  • Hover states on certain buttons
  • Links and important UI indicators
Regular buttons use white/grey backgrounds to avoid overwhelming the interface. Only important interactive elements get the purple highlight for better usability.

Can this affect your data after installing the module?

Doesn't affect your data - only the visual appearance changes.

Can I use this theme for only specific users or companies in multi-company setup?

No, Odoo themes apply globally to all users and companies because they modify the web.assets_backend bundle. If you need per-user or per-company themes, you would need custom development to implement user-based CSS switching, which is not a standard Odoo feature. The theme applies system-wide once installed, affecting everyone using that Odoo instance.
